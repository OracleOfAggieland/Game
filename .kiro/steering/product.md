# Product Overview

This is a React-based Snake game with both single-player (Classic) and multiplayer (Arena) modes. The game features AI opponents, real-time multiplayer capabilities through Firebase, and is currently being enhanced with power-up systems, progressive wave-based gameplay, and performance optimizations.

## Key Features
- Classic single-player Snake gameplay
- Arena multiplayer mode with AI opponents
- Real-time synchronization via Firebase
- Progressive wave system with boss encounters
- Power-up system with strategic elements
- Mobile-responsive design

## Target Experience
The game aims to provide smooth 60 FPS gameplay with polished visual effects, immediate Arena mode entry, and challenging progressive difficulty that keeps players engaged through strategic power-up collection and wave-based survival mechanics.
